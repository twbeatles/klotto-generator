# 🎰 Lotto 6/45 Generator Pro v2.1

**Lotto Generator Pro**는 파이썬과 PyQt6로 제작된 현대적이고 강력한 로또 번호 생성기입니다. 동행복권 공식 API를 연동하여 실시간 당첨 정보를 제공하며, 사용자 맞춤형 옵션과 분석 알고리즘을 통해 최적의 행운 번호를 제안합니다.

![Screenshot](screenshot.png)

## ✨ 주요 기능

### 1. 강력한 번호 생성
- **고정수/제외수**: 반드시 포함하거나 제외할 숫자를 지정할 수 있습니다.
- **범위 입력 지원**: `1-10, 20, 30-35` 형식으로 편리하게 입력 가능합니다.
- **연속수 제한**: 연속된 숫자가 나오지 않도록 제한하여 확률을 높입니다.
- **대량 생성**: 한 번에 최대 20게임까지 생성 가능합니다.

### 2. 스마트 분석 시스템
- **자동 분석**: 생성된 번호의 합계, 홀짝 비율을 실시간으로 계산하여 표시합니다.
- **최적 조합 추천**: 통계적으로 당첨 확률이 높은 구간(합계 100~175)을 시각적으로 안내합니다.
- **당첨 확인**: 생성된 번호가 지난 당첨 번호와 일치하는지 즉시 비교할 수 있습니다.

### 3. 📊 번호 통계 (NEW!)
- **출현 빈도 분석**: 가장 많이/적게 나온 번호를 한눈에 확인합니다.
- **번호대별 분포**: 1-10, 11-20, 21-30, 31-40, 41-45 구간별 분포를 분석합니다.
- **생성 히스토리 기반**: 내가 생성한 번호들의 패턴을 파악할 수 있습니다.

### 4. 📜 생성 히스토리 (NEW!)
- **자동 저장**: 생성한 모든 번호 조합을 자동으로 기록합니다.
- **최대 500개**: 최근 500개의 조합을 날짜와 함께 보관합니다.
- **중복 감지**: 이전에 생성한 조합과 중복되면 자동으로 감지합니다.

### 5. 사용자 친화적 UI/UX
- **다크 모드**: 눈이 편안한 다크 모드와 깔끔한 라이트 모드를 지원합니다.
- **3D 로또 공**: 입체감 있는 현대적인 로또 공 디자인
- **즐겨찾기**: 마음에 드는 번호 조합을 별도로 저장하고 관리할 수 있습니다.
- **편의 기능**: 행별 복사, 파일 저장, 이전 회차 조회 등 다양한 편의 기능을 제공합니다.

## 🛠️ 설치 및 실행

### 요구 사항
- Python 3.8 이상
- PyQt6 라이브러리

### 설치
```bash
pip install PyQt6
```

### 실행
```bash
python klottogenerator.py
```

### 빌드 (실행 파일 생성)
```bash
pip install pyinstaller
pyinstaller klottogenerator.spec
```
빌드된 실행 파일은 `dist/LottoGeneratorPro/` 폴더에 생성됩니다.

## ⌨️ 단축키

| 단축키 | 기능 | 설명 |
|---|---|---|
| **Ctrl + G** | 번호 생성 | 설정된 옵션으로 새 번호를 생성합니다. |
| **Ctrl + R** | 초기화 | 생성된 결과를 모두 지웁니다. |
| **Ctrl + S** | 저장 | 결과를 텍스트 파일로 저장합니다. |
| **Ctrl + D** | 테마 전환 | 다크 모드와 라이트 모드를 전환합니다. |

## 📁 데이터 저장 경로
사용자의 데이터는 홈 디렉토리의 숨김 폴더에 안전하게 저장됩니다.

```
~/.lotto_generator/
├── app.log           # 애플리케이션 로그
├── favorites.json    # 즐겨찾기 데이터
└── history.json      # 생성 히스토리
```

- Windows: `C:\Users\Username\.lotto_generator\`
- macOS/Linux: `~/.lotto_generator/`
